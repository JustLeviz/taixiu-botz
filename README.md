# TaiXiu Discord Bot (Render-ready)

## Requirements
- Node.js >= 18
- DISCORD_TOKEN env var
- (Render) Persistent Disk recommended

## Render setup
- Build Command: npm install
- Start Command: npm start

Env Vars:
- DISCORD_TOKEN = <your bot token>
- DB_PATH = /var/data/bot.db   (recommended on Render)

Persistent Disk:
- Mount path: /var/data
